## ✨ Features

* **Home Page:** Overview of the school, mission, and vision.
* **About Us:** Detailed history and core values of the institution.
* **Academics:** Information regarding the curriculum, courses, and educational approaches.
* **Infrastructure:** Details about the school's facilities, labs, and resources.
* **Management:** Profiles of the school's leadership and administration.
* **Gallery:** A visual collection of events, sports meets, and achievements.
* **Contact:** Direct channels to reach out to the school administrative office.
* **Responsive Design:** Completely optimized for viewing on desktops, tablets, and smartphones.

## 🛠️ Tech Stack

* **Front-end:** HTML5, CSS3, Vanilla JavaScript
* **Backend:** Node.js, Express (via `server.js`)
* **Dependencies:** npm (for local packages and scripts)

## 🚀 Local Development Setup

If you want to run this website locally on your machine, follow these steps:

1. **Clone the repository** (or download the files).
2. Ensure you have [Node.js](https://nodejs.org/) installed on your machine.
3. Open a terminal to the project directory:
   ```bash
   cd "Hayagrivas published final\Hayagrivas published final"
   ```
4. **Install dependencies**:
   ```bash
   npm install
   ```
5. **Start the local server**:
   ```bash
   node server.js
   ```
   *or*
   ```bash
   npm start
   ```
6. Open your web browser and navigate to `http://localhost:3000` (or whichever port is defined in `server.js`).

## 📁 Project Structure

* `*.html` - Core pages (index, about, academics, etc.)
* `style.css` / `responsive.css` / `modal.css` - Styling files for layout and responsiveness.
* `script.js` - Client-side interactivity and logic.
* `server.js` - Node.js server setup.
* Images/Assets - Various `.png` and `.jpg` resources for the frontend UI.
* `package.json` - Node dependencies and scripts.
